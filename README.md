# Rol - Website Studio Film Analog

Landing page untuk **Rol**, studio cuci dan scan film analog di Indonesia. Website ini memperkenalkan layanan Rol melalui tampilan yang terinspirasi dari suasana kamar gelap, lengkap dengan informasi proses, harga, galeri, testimoni, dan kontak.

Website dibuat sebagai halaman statis menggunakan HTML, CSS, dan JavaScript tanpa framework atau backend.

## Lihat Website

Versi website yang sudah dirender dapat dibuka melalui GitHub Pages:

### [Buka Website Rol](https://lambert-lincoln.github.io/ai-website/)

Apabila tautan belum dapat dibuka, berarti deployment GitHub Pages masih dalam proses atau belum diaktifkan pada repository ini.

## Unduh Source Code

Untuk mengunduh seluruh source code tanpa menggunakan Git:

1. Buka halaman repository ini.
2. Klik tombol **Code**.
3. Pilih **Download ZIP**.

Atau gunakan [tautan unduh ZIP langsung](https://github.com/lambert-lincoln/ai-website/archive/refs/heads/main.zip).

Untuk melakukan clone dengan Git:

```bash
git clone https://github.com/lambert-lincoln/ai-website.git
cd ai-website
```

## Menjalankan Secara Lokal

Karena website ini bersifat statis, buka `index.html` langsung di browser. Alternatifnya, jalankan server lokal:

```bash
python3 -m http.server 8000
```

Kemudian buka [http://localhost:8000](http://localhost:8000) di browser.

## Struktur Repository

- `index.html` - halaman website beserta CSS dan JavaScript.
- `design.md` - spesifikasi dan arahan desain website.
- `README.md` - dokumentasi repository.
